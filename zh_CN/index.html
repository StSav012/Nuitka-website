Nuitka 是什么

Nuitka 是用 Python 编写的优化 Python 编译器，它可以创建不需要单独安装程序就能运行的可执行文件。数据文件既可以包含，也可以放在旁边。

它完全兼容 Python2（2.6, 2.7）和 Python3（3.3 - 3.10），可以在 Windows, macOS, Linux 等操作系统上运行，基本上 Python 已经可以为你工作了。

它保护你的代码和数据，特别是与 Nuitka 的商业提供，也只是可工作和易于使用。

Nuitka 是在 Apache 许可下免费发布的。Nuitka commercial 是一个私有的存储库，你可以付费访问并获得用于商业用途的插件。

与此同时，你可以检查为什么 Nuitka commercial 是为你或找到它的 标准版本在这里。
