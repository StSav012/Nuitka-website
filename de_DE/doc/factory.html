Fabrikanweisungen

Für was ist die Fabrik

Die master und develop Zweige sollte jederzeit funktionieren. Für Experimente die Probleme hervorrufen könnten, wie zum Beispiel neue Features ist der factory Zweig geeignet.

Du bist warscheinlich hier, weil eine Änderung für sie vorgenommen wurde.

Wie bekommt man factory

git

git clone --branch factory https://github.com/Nuitka/Nuitka.git

pip

pip install -U --force-reinstall "https://github.com/Nuitka/Nuitka/archive/factory.zip"

Wort der Warnung

Dieser Zweig kann alle Arten von dummen Fehlern enthalten. Zum Beispiel ist es möglich, dass Nuitka nicht mit allen Python Versionen funktioniert, crasht, oder überhaupt nicht funktioniert. Hier werden auch Änderungen ohne Hinweise vorgenommen. Sie sollten ihn nur benutzen wenn sie ihn wirklich benötigen.

Wenn die Änderungen für gut befunden werden, werden sie relativ schnell in den develop Zweig übernommen. Bei einem Fix sollten sie warten bis er dort verfügbar ist.
