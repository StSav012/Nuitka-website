Verwendung

Damit wird die Nuitka-Website aufgebaut. Erweiterungen aller Art sind willkommen. Du brauchst Python 3.9 und pipenv, um es zu bauen, und bis jetzt wurde es nur unter Linux gemacht, aber es sollte auch unter Windows funktionieren, du kannst das als Teil deines Beitrags machen.

# Creates a virtualenv with all that is needed to develop the
# site.
python3 -m pip install -U invoke
python3 -m invoke virtualenv

# Now lets build the site, to see if it's all correct.
python3 -m invoke doc

# Optionally build the localized sites.
python3 -m invoke intl -l zh_CN
python3 -m invoke intl -l de_DE
...

# Post processing, e.g. adds language switcher, compact
# JS/CSS, adapts external links, and more.
python3 -m invoke run -t post-process

# Start local web server with the site, and do automatic
# rebuilds. Localization are rebuilt if they were previously
# built.
python3 -m invoke run -t serve-site

# Browse it (Linux)
xdg-open http://localhost:8080
# Browse it (Windows)
explorer http://localhost:8080

Verwendung von Sphinx

Die Seite ist also im Grunde eine Automatisierung des Imports einiger Dateien aus dem Nuitka git repository, die z.B. die Changelog.rst in Seiten aufteilt und ansonsten Sphinx zum Rendern verwendet. Siehe die Sphinx-Dokumentation, insbesondere die für das Thema Read the Docs und ABlog.

Wie wird übersetzt

# Generate the .pot files
cd intl
make gettext

# Generate .po files from .pot
sphinx-intl update -p ../output/gettext -l your_language_code

Unter locales in dem Ordner mit dem Sprachkürzel sind viele .po Dateien. In der .po Datei kannst du msgid und darunter msgstr sehen. In msgid befindet sich das englische Original, in msgstr kannst du die Übersetzung reinschreiben. Bitte committe nur die Dateien, die du übersetzt hast, wenn du deine Änderungen per PR hochlädst.

Bild Optimierung

# Optimize PNG files like this, normally not needed, this
# is lossless.
sudo apt-get install optipng
find . -iname *.png -a -type f -exec optipng -o7 -zm1-9 {} \;

# Optimize JPEG files like this, normally not needed, this
# is lossless.
sudo apt-get install jpegoptim
find . -iname *.jpg -a -type f -exec jpegoptim {} \;
